# local

```
开发平台：Windows
开发工具：webstorm+nodejs10+MySql5.7+vue
技术实现：nest（IOC框架）+element-ui+bootstrap+vue
项目描述：该系统采用webstorm开发工具、mysql数据库、nest框架、bootstrap4、axios技术、vue框架，对网站进行了设计。
功能实现：1、后台管理功能，用户管理、城市管理、文章管理
		2、用户管理，增删改查用户
		3、城市管理，增加城市
	4、文章管理，对几大模块添加文章，并上传图片等附件
责任描述：个人独立完成项目设计、开发、测试。该系统基于nest框架，前端使用vue框架、axios技术。

```



## QuickStart



### Development

```bash
fronted是vue项目
local是后端nest项目。
进入fronted之后
$ npm i
$ npm run dev
$ open http://localhost:8081/

进入local之后
$ npm i
$ npm run start
$ open http://localhost:8080/


```
